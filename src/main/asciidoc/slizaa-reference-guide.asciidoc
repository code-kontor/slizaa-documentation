== Preface

This is the reference documentation of Slizaa, an interactive, web-based tool for analyzing and visualizing the dependencies of java-based software systems.
This guide covers all the functionality provided by Slizaa.

In case this guide doesn't answer all your questions just join the Slizaa https://groups.google.com/forum/#!forum/slizaa-users[Google group] to get help. We also monitor the Slizaa tag on StackOverflow.

You found a typo or other error in this guide? Please let us know by opening an issue in the https://github.com/code-kontor/slizaa-documentation[Slizaa Documentation GitHub repository],
or, better yet, help the community and send a pull request for fixing it!
